Install NetShield

Works with Chromium-based browsers (Chrome, Edge, Brave, etc.)

Download NetShield (.zip)
1

Download & extract

Click the button above to download the extension. Extract the ZIP file to a folder on your computer.

2

Open Extensions page

In Chrome, go to chrome://extensions and turn on Developer mode (top right).

3

Load the extension

Click Load unpacked, then select the NetShield-Extension folder containing manifest.json.

The extension icon will appear in your browser toolbar. Click it to open the NetShield popup.
